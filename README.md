# Web Fingerprinting Tool

Passive HTTP-based fingerprinting tool for identifying visible web technologies from owned or authorized targets.

## Features
- URL or domain input
- HTTP header analysis
- Basic technology detection
- Security header overview
- Cookie inspection
- Redirect chain summary
- Response time measurement
